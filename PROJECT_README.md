## 🌐 Projeto de Portfólio / Portfolio Project

### Português (pt/🇧🇷)

**Este projeto de portfólio foi criado com a intenção de ser útil para todos os desenvolvedores. Ele ainda está em desenvolvimento, com muitas funcionalidades planejadas para serem adicionadas. O projeto foi feito inteiramente em Laravel 11.**

### Funcionalidades atuais:
- Suporte a dois idiomas (pt-BR e en-US).
- Adição de informações pessoais e logos.
- Configuração de variáveis de ambiente (.env).
- Registro de usuários.
- Adição de projetos em dois idiomas.
- Execução de demonstrações de projetos em PHP, Laravel e HTML.
- Adição de habilidades (atualmente, apenas linguagens e frameworks).
- Inclusão de projetos com imagens e vídeos.
- Monitoramento de visitantes (diariamente, mensalmente, e anualmente).
- Personalização de cores e temas.

### Funcionalidades futuras:
- Integração com API do Google Analytics.
- Expansão das habilidades adicionadas.
- Configuração de frases no site.
- Melhorias no sistema de registro de usuários.
- Adição de auto-instalação de projetos.

⚠ **Observação**: O painel de controle atualmente está disponível apenas em pt-BR, mas a versão em en-US será lançada em breve. O design está em fase inicial e será melhorado com opções de temas para o painel e página inicial. O foco atual é a implementação de funcionalidades principais na página principal.

Este projeto tem como objetivo melhorar minhas habilidades com Laravel e design, além de servir como uma base para quem deseja criar seu próprio portfólio. **Sinta-se à vontade para usar este projeto, mas peço que me dê os devidos créditos caso o utilize.**

### English (en/🇺🇸)

**This portfolio project was created with the intention of being usable for all developers. It is still under development, with many features planned for future additions. The project was built entirely with Laravel 11.**

### Current Features:
- Support for two languages (pt-BR and en-US).
- Add personal information and logos.
- Configure environment variables (.env).
- User registration.
- Add projects in two languages.
- Demo execution of projects in PHP, Laravel, and HTML.
- Add skills (currently only languages and frameworks).
- Add projects with images and videos.
- Track visitors (daily, monthly, and annually).
- Customize colors and themes.

### Upcoming Features:
- Google Analytics API integration.
- Add more skills.
- Configure phrases on the site.
- Improve user registration.
- Add project auto-installation.

⚠ **Note**: The control panel is currently only available in pt-BR, but an en-US version will be released soon. The design is still simple and will be improved with theme options for both the panel and homepage. The current focus is on the main page's core features.

This project aims to enhance my Laravel and design skills while serving as a base for anyone looking to create their own portfolio. **Feel free to use this project, but I kindly ask that you give me credit if you use it.**
